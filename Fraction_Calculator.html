How to Use

  • Enter Fractions: Input the numerator and denominator for both fractions in the provided fields.
  • Select Operation: Choose the mathematical operation (+, −, ×, ÷) you want to perform.
  • Calculate: Click the "Calculate" button to see the result in both fractional and decimal form.
  • Responsive Design: This calculator works perfectly on both desktop (Windows) and mobile (Android) devices.
  • Fraction Rules: The calculator automatically simplifies results to their lowest terms.

Note: The calculator will display an error if you try to divide by zero or use invalid inputs. All calculations are performed in real-time.
